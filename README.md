# 😷 Face Mask Detection using Deep Learning

## 📌 Problem Statement
The goal of this project is to build a deep learning model that can detect whether a person is wearing a face mask or not. This solution can be useful in public safety systems, especially during health crises like COVID-19.

---

## 📊 Dataset
- Image dataset containing two classes:
  - With Mask 😷
  - Without Mask 😐  
- Dataset includes labeled images for training and testing the model

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Libraries & Frameworks:**  
  - NumPy  
  - Pandas  
  - OpenCV  
  - TensorFlow / Keras  
  - Matplotlib  

---

## ⚙️ Project Workflow

1. **Data Collection**
   - Gathered labeled images of masked and unmasked faces  

2. **Data Preprocessing**
   - Resized images  
   - Normalized pixel values  
   - Converted into suitable format for model training  

3. **Model Building**
   - Used Convolutional Neural Network (CNN)  
   - Designed layers for feature extraction and classification  

4. **Model Training**
   - Trained model on training dataset  
   - Validated using test dataset  

5. **Model Evaluation**
   - Evaluated accuracy and performance  
   - Checked predictions on sample images  

---

## 📈 Results
- Achieved good accuracy in detecting masked vs unmasked faces  
- Model performs well on test data and sample inputs  

---

## 📷 Output
- Detects face mask status:
  - ✅ With Mask  
  - ❌ Without Mask
